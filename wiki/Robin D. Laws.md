# Robin D. Laws

## Resumo
**Robin D. Laws** é um designer de RPG, autor do sistema **GUMSHOE** e de jogos como *Hillfolk* e *Dying Earth*. Foi convidado por [[Greg Stafford]] para criar o **Hero Wars** (posteriormente [[HeroQuest]]), um *storytelling game* para narrativas colaborativas em [[Glorantha]].

## Conteúdo

### Trabalhos Notáveis
- **GUMSHOE** — Sistema de RPG focado em investigação
- **Hillfolk** — Jogo de drama interpersonal (usa sistema DramaSystem)
- **Dying Earth** — RPG baseado nos romances de Jack Vance
- **Hero Wars** (2000) — Criado a pedido de Greg Stafford para Glorantha

### Contribuição para Glorantha
A pedido de [[Greg Stafford]], Robin D. Laws desenvolveu o **Hero Wars** — um *storytelling game* projetado para facilitar a colaboração de jogadores na criação de histórias em Glorantha. Este sistema evoluiria para o [[HeroQuest]] e, eventualmente, para o [[QuestWorlds]].
