# Ian Cooper

## Resumo
**Ian Cooper** é o designer liderando as transformações do sistema HeroQuest em [[QuestWorlds]]. Tem interesse em publicar suplementos não-Glorantha para o sistema, buscando posicioná-lo como agnóstico de cenário, apesar de suas origens intimamente ligadas a [[Glorantha]].

## Conteúdo

### Papel no QuestWorlds
Ian Cooper liderou as profundas mudanças no sistema HeroQuest que resultaram no [[QuestWorlds]], incluindo a criação do SRD disponível no GitHub da Chaosium.

### Visão para o Sistema
Cooper deseja que o QuestWorlds seja **agnóstico de cenário**, não apenas ligado a Glorantha. Para isso, planeja publicar suplementos para outros cenários. Esta abordagem pode estar atrasando o lançamento de **QuestWorlds: Glorantha**.

### Conexões
- Responsável pela evolução: HeroQuest → QuestWorlds
- Ligado à [[Chaosium]]
- Histórico com [[Glorantha]] e [[HeroQuest Glorantha]]
